# UI Автоматизация: Тинькофф Журнал

Репозиторий содержит автотесты для проверки ключевого функционала Тинькофф Журнала. Проект построен на базе **Playwright** с использованием паттерна **Page Object Model (POM)**.

## 🔍 Объекты тестирования
*   **Навигационное меню**: Проверка кликабельности разделов и переходов.
*   **Модуль поиска**: Интеграция поисковых запросов и отображение и проверка результатов.
*   **Калькулятор дохода**: Проверка полей, e2e результат среднедушевого дохода.

---

## 🛠 Стек технологий
*   **Язык**: Python 3.10+
*   **Фреймворк**: Pytest
*   **Библиотека**: Playwright
*   **Браузер**: Chromium (Chrome)

---

## ⚙️ Установка и настройка

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <ссылка_на_репозиторий>
    cd <папка_проекта>
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python -m venv venv
    # Windows
    venv\Scripts\activate
    # macOS/Linux
    source venv/bin/activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Установите драйвер Chromium:**
    ```bash
    playwright install chromium
    ```

---

## 🚀 Запуск тестов

Тесты запускаются через `pytest`. 

*   **Запуск всех тестов :**
    ```bash
    pytest
    ```
